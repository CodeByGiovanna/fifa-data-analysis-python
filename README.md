# FIFA Data Analysis Python

## Descrição

Projeto acadêmico desenvolvido em Python utilizando Pandas, SQLAlchemy e DuckDB para análise de dados de jogadores do FIFA.

O trabalho inclui:

* Importação e tratamento de dados CSV
* Filtragem de jogadores âncora
* Comparação entre diferentes versões do FIFA
* Consultas SQL em banco de dados em memória
* Análises estatísticas e consolidação de dados
* Relatório longitudinal FIFA 15 a FIFA 21

## Tecnologias Utilizadas

* Python
* Pandas
* SQLAlchemy
* DuckDB
* Deepnote
